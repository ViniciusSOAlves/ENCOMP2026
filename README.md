Pré requisitos para funcionar: ter o NodeJS instalado em seu computador e um servidor MySQL, recomenda-se o WampServer, o mesmo que foi utilizado para o desenvolvimento.

Após a clonagem da pasta rodar o comando "npm i" na pasta geral e tambem dentro da pasta backend. Para iniciar o site, rodar o comando "npm run dev" na pasta geral
e também na backend.
